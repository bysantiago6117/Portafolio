Máquina de Turing - Simulación Interactiva

Este proyecto es una simulación interactiva de una Máquina de Turing, un dispositivo teórico que puede manipular y procesar datos siguiendo reglas específicas. Permite comprender los fundamentos de la computación y los algoritmos. La simulación se desarrolló utilizando JavaScript para la lógica de la Máquina de Turing y HTML, CSS y Bootstrap para el diseño y la organización del sitio web.

Autómata Finito - Representación Gráfica

Este proyecto académico representa visualmente un autómata finito utilizando un grafo. El objetivo es ilustrar su funcionamiento y comprender los conceptos clave de la teoría de autómatas. La representación gráfica del autómata se desarrolló utilizando JavaScript para la lógica y programación del grafo. Además, se utilizó HTML, CSS y Bootstrap para el diseño y la estructura del sitio web.

Tecnologías utilizadas:

Estaciones - Registro y Visualización en Mapa

  • Resumen del proyecto: Se desarrolló una página web para registrar y mostrar estaciones en un mapa. Cada estación tiene parcelas asociadas y se visualiza un icono en el mapa.
  • Backend: Se utilizó Spring Boot con las siguientes dependencias:
    • spring-boot-starter-data-jpa: Para la comunicación con la base de datos y la implementación de operaciones CRUD.
    • spring-boot-starter-web: Para la creación de servicios web.
    • postgresql: Para la integración con la base de datos PostgreSQL.
  • Frontend: Se utilizó React junto con varios liberias adicionales:
    • React Router para el enrutamiento de la aplicación.
    • React Bootstrap para la creación de componentes y estilos.
    • React Leaflet para la integración de mapas interactivos.
    • Axios para realizar peticiones HTTP al backend.
Tecnologías utilizadas:

Vuelos - Reserva y Búsqueda de Vuelos

  • Resumen del proyecto: Aplicación de reserva y búsqueda de vuelos. Permite a los usuarios registrarse, iniciar sesión y realizar acciones como crear vuelos, buscar vuelos según criterios de búsqueda y realizar reservas.
  • Backend: Se utilizó Spring Boot con las siguientes dependencias:
    • spring-boot-starter-data-jpa: Para la comunicación con la base de datos y la implementación de operaciones CRUD.
    • jjwt: Para la generación y verificación de tokens JWT (JSON Web Tokens).
    • spring-boot-starter-security: Para la seguridad del sistema, se utilizó Spring Security con JWT.
    • postgresql: Para la integración con la base de datos PostgreSQL.
  • Frontend: Se utilizó React junto con varias librerías adicionales:
    • React Router para el enrutamiento de la aplicación.
    • React Bootstrap para la creación de componentes y estilos.
    • Axios para realizar peticiones HTTP al backend.
